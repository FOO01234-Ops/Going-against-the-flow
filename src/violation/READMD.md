逆行检测核心算法

┌─────────────────────────────────────────────────────────────────────────────────────────┐
│ 逆行检测算法流程 │
├─────────────────────────────────────────────────────────────────────────────────────────┤
│ │
│ ┌──────────────────────────────────────────────────────────────────────────────────┐ │
│ │ 轨迹点收集 │ │
│ │ (连续帧中同一车辆的质心坐标) │ │
│ │ [(x₁,y₁), (x₂,y₂), ..., (xₙ,yₙ)] │ │
│ └──────────────────────────────────────────────────────────────────────────────────┘ │
│ │ │
│ ▼ │
│ ┌──────────────────────────────────────────────────────────────────────────────────┐ │
│ │ 方向计算 │ │
│ │ 起点 → 终点 位移向量 (dx, dy) │ │
│ │ 角度 θ车 = atan2(dy, dx) × 180/π │ │
│ └──────────────────────────────────────────────────────────────────────────────────┘ │
│ │ │
│ ▼ │
│ ┌──────────────────────────────────────────────────────────────────────────────────┐ │
│ │ 车道匹配 │ │
│ │ 读取摄像头对应车道预设方向角度 θ道 │ │
│ │ 支持: 多摄像头 / 分叉车道 / 多车道 │ │
│ └──────────────────────────────────────────────────────────────────────────────────┘ │
│ │ │
│ ▼ │
│ ┌──────────────────────────────────────────────────────────────────────────────────┐ │
│ │ 角度差判定 │ │
│ │ Δθ = |θ车 - θ道| │ │
│ │ Δθ > 90° → 逆行 │ │
│ │ Δθ ≤ 90° → 正常 │ │
│ └──────────────────────────────────────────────────────────────────────────────────┘ │
│ │
└─────────────────────────────────────────────────────────────────────────────────────────┘
