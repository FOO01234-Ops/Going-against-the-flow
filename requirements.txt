环境要求
Python 3.9 或更高版本

CUDA 11.8+ (可选，GPU加速)

8GB+ RAM (推荐 16GB)

安装步骤
# 1. 克隆项目
git clone <repository-url>
cd Going\ against\ the\ flow

# 2. 创建虚拟环境 (conda)
conda create -n yolo26_env python=3.9
conda activate yolo26_env

# 3. 安装依赖
pip install -r requirements.txt
